# Episodio 01: Welcome Aboard

## Resumen

En este episodio se presentó el curso **Laravel From Scratch 2026** y se explicó el objetivo general del proyecto. El propósito principal fue comprender que el curso se desarrollará de forma progresiva, construyendo una aplicación Laravel mientras se aplican fundamentos del framework, buenas prácticas de organización, rutas, vistas, formularios, bases de datos, autenticación y otros temas importantes del desarrollo web moderno.
